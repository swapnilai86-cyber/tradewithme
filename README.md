# Algo Swing Trading Platform

A complete, production-ready algorithmic swing trading platform for Indian equities (Nifty 500).

## Tech Stack
- **Backend:** FastAPI, Python 3.11, SQLAlchemy, Alembic
- **Frontend:** React (Vite), Vanilla CSS (Premium Glassmorphism)
- **Database:** PostgreSQL 15
- **Broker:** m.Stock API integration
- **Alerts:** Discord Webhooks
